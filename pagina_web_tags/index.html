História da Web

A história da Web é uma jornada fascinante que revolucionou a forma como nos comunicamos, acessamos informações e interagimos com o mundo. Navegadores e HTML são peças-chave nessa narrativa, moldando a experiência online desde os primórdios.

A Internet nasceu como uma rede militar nos Estados Unidos, com o objetivo de conectar universidades e centros de pesquisa. No entanto, a forma como conhecemos a Web hoje só se tornou possível graças à criação da World Wide Web (WWW).

O físico inglês Tim Berners-Lee, trabalhando no CERN (Organização Europeia para a Pesquisa Nuclear), desenvolveu em 1989 o HTML (HyperText Markup Language). Essa linguagem de marcação permitiu a criação de documentos hipertextuais, ou seja, textos com links que levavam a outros documentos.

Berners-Lee também criou o primeiro navegador, chamado WorldWideWeb. Esse software simples permitia visualizar as primeiras páginas HTML.

O navegador Mosaic, lançado em 1993, foi um marco importante. Ele popularizou a Web, tornando-a acessível a um público mais amplo. O Mosaic introduziu conceitos como imagens incorporadas e suporte a diferentes tipos de mídia.

Inspirado no Mosaic, o Netscape Navigator se tornou o navegador dominante nos meados dos anos 90. Essa competição impulsionou o desenvolvimento de novos recursos e padrões para a Web.

A disputa entre Netscape e Microsoft (com o Internet Explorer) deu origem à chamada "guerra dos navegadores". Essa rivalidade acelerou a evolução da Web, com a introdução de novas tecnologias e funcionalidades.

Para garantir a interoperabilidade entre diferentes navegadores e plataformas, foi criado o Consórcio W3C (World Wide Web Consortium). O W3C é responsável por definir os padrões da Web, como HTML, CSS (Cascading Style Sheets) e JavaScript.

A última grande atualização do HTML, o HTML5, trouxe novos elementos e funcionalidades, como vídeo, áudio, gráficos vetoriais e geolocalização. Essa versão tornou a Web mais rica e interativa.

Hoje em dia, temos uma variedade de navegadores modernos, como Chrome, Firefox, Safari e Edge. Esses navegadores são altamente otimizados e oferecem uma experiência de navegação rápida e segura.

O que são Tags HTMl

Tags HTML são como marcadores que um navegador da web usa para entender e exibir o conteúdo de uma página da internet. Elas são os blocos de construção básicos do HTML (HyperText Markup Language), a linguagem que define a estrutura e o conteúdo das páginas web.

Tag html

A tag html delimita todo conteúdo de um documento HTML, abrangendo todas as demais tags. O código HTML inicia coma tag html e termina com /htlm.

(Voltar)

Tag head

A tag head é o cabeçalho, responsável por fornecer ao navegador informações sobre a página web que não serão exibidas para os usuários. O cabeçaçho inicia com head e termina com /head

(Voltar)

Tag body

A tag body é responsável pelo corpo do documento HTML. É nessa tag que está contido todo o conteúdo que aparecerá na página web. A tag inicia com body e termina com /body, e todo o conteudo que estiver entre o inicio e o fim da tag aparecerá para o usuário.

(Voltar)

Tag h

A tag h serve para formatar o texto como um título. Essa tag possui variações (h1, h2, h3, h4, h5 e h6) que são os formatos de subtítulos. A tag inicia com h e termina com /h, e respectivamente para cada uma de suas variações.

(Voltar)

Tag p

A tag p serve para formatar o texto como parágrafo. A tag inicia com p e termina com /p, e todo o conteúdo que estiver entre o inicio e final da tag irá aparecer na página web como um parágrafo.

(Voltar)

Tag blockquote

A tag blockquote serve para formatar o texto como citação, deixando uma margem a esquerda do texto. A tag inicia com blockquote e termina com /blockquote, e todo o conteúdo dentro dela aparecerá formatado como citação.
(Voltar)

Tag ol

A tag ol serve para criar uma lista ordenada de itens. A tag começa com ol e termina com /ol, devendo possuir tags li dentro dela para formar os itens da lista.

(Voltar)

Tag ul

A tag ul serve para criar uma lista não ordenada de itens. A tag começa com ul e termina com /ul, devendo possuir tags li dentro dela para formar os itens da lista.

(Voltar)

Tag li

A tag li serve para adicionar os itens das listas, tanto ordenadas, quanto as listas não ordenadas. A tag começa com li e termina com /li, e seu conteúdo aparece como um item da lista da qual pertence.

(Voltar)

Tag a

A tag a serve para inserir links e possibilitando a navegação através das páginas web. A tag inicia com a e termina com /a e deve ser indicado no seu corpo o caminho para qual deseja ir através do atriburo href e o nome do link que aparecerá para o usuário.

(Voltar)

Tag input

A tag input serve para adicionar um campo interativo com o usuário. É uma tag única que inicia com input e não precisa ser terminada, os atributos input são definidos dentro da própria tag que inicia.

(Voltar)

Tag img

A tag img serve para adicionar uma imagem na sua página web. É uma tag única que inicia com img e não precisa ser terminada, os atributos img são definidos dentro da própria tag que inicia.

(Voltar)

Tag strong

A tag strong serve para fazer marcações em textos, tranformando letras, palavras ou frases em negrito. A tag inicia com strong e termina com /strong.

(Voltar)

Tag i

A tag i serve para fazer marcações em textos, tranformando letras, palavras ou frases em itálico. A tag inicia com i e termina com /i.

(Voltar)

Tag u

A tag u serve para fazer marcações em textos, sublinando letras, palavras ou frases. A tag inicia com u e termina com /u.

(Voltar)

Tag sup

A tag sup serve para fazer marcações em textos, sobrescrevendo letras, palavras ou frases acima da linha normal. A tag inicia com sup e termina com /sup.

(Voltar)

Tag mark

A tag mark serve para fazer marcações em textos, destacando letras, palavras ou frases. A tag inicia com mark e termina com /mark.

(Voltar)

Tag small

A tag small serve para deixar as letras do texto com um tamanho menor que o normal. A tag inicia com small e termina com /small.

(Voltar)
